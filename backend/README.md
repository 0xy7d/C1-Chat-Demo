## A Quick implementation for AI-Assistant

This is a quick implementation for AI-Assistant. The goal was to try and understand how the C1 Chat Ui works.

You can check out the documentation [here](https://platform.openai.com/docs/guides/chat/introduction)

## Usage

- Run `python main.py`

## Requirements

- `python -m pip install -r requirements.txt`

or using uv:

- `uv pip install -r requirements.txt`


#### Some artifacts and commands for future references

- `uv pip compile pyproject.toml -o requirements.txt --no-annotate --no-header`
